
# Recrear diseño en Figma

Este es un paquete de código para Recrear diseño en Figma. El proyecto original está disponible en https://www.figma.com/design/L6j75TFCnFWrqplsrXaSb0/Recreate-Design-in-Figma.

## Ejecutar el código

Ejecuta `npm i` para instalar las dependencias.

Ejecuta `npm run dev` para iniciar el servidor de desarrollo.
